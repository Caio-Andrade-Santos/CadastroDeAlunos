Cadastro de Alunos — C# / Windows Forms
 Descrição

Aplicação desktop desenvolvida em C# utilizando Windows Forms, com o objetivo de simular um sistema de cadastro de alunos.
O sistema permite cadastrar, editar, excluir e listar alunos, aplicando validações de CPF, RG e RA antes do salvamento.

 Funcionalidades

Cadastro, edição e exclusão de alunos (CRUD)

Validação de CPF, RG e RA

Persistência de dados em banco de dados local (dbAlunos)

 Tecnologias Utilizadas

C#

Windows Forms

Banco de Dados Local

Programação Orientada a Objetos (POO)

 Objetivo do Projeto

Praticar conceitos de POO, validação de dados e integração com banco de dados em aplicações desktop.
